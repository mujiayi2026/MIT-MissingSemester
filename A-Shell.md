# Shell

## `ls -l` 命令详解

### -l 选项的作用

`ls -l`（或 `ls --long`）用于显示文件的详细信息。根据 `man ls` 的说明，其功能为 Display detailed information。

执行 `ls -l /` 命令后，会列出根目录下所有文件的详细信息，每行输出以 10 个字符开头。

### 文件信息行的字符含义

每行最前面的 10 个字符的含义如下：

```
d rwx r-x r-x
│  │   │   │
│  │   │   └── 其他用户权限：r-x（可读、可执行、不可写）
│  │   └────── 所属组权限：r-x（可读、可执行、不可写）
│  └────────── 所有者权限：rwx（可读、可写、可执行）
└───────────── 文件类型：d（目录）
```

**字符划分说明：**

- **第 1 个字符**：文件类型（如 `d` 表示目录，`-` 表示普通文件）
- **第 2-10 个字符**：文件权限（共 9 个字符，分为 3 组）
  - 第 2-4 个字符：文件所有者的权限
  - 第 5-7 个字符：文件所属组的权限
  - 第 8-10 个字符：其他用户的权限

每组权限由 3 个字符组成，分别表示读（r）、写（w）、执行（x）的权限，无该权限时用 `-` 表示。

## Glob 模式详解

### 什么是 Glob？

Glob 是 shell 中用于匹配文件名的一种模式匹配语法。例如在命令 `find ~/Downloads -type f -name "*.zip" -mtime +30` 中，`*.zip` 就是一个 glob 模式。

### Glob 的常见语法

根据 `man bash` 中的 Pattern Matching 部分，Glob 支持以下基本语法：

| 模式 | 说明 | 示例 |
|------|------|------|
| `*` | 匹配任意数量的任意字符 | `ls *.txt` 匹配所有 `.txt` 文件 |
| `?` | 匹配单个任意字符 | `ls file?.txt` 匹配 `file1.txt`、`fileA.txt` 等 |
| `[abc]` | 匹配括号内任意一个字符 | `ls file[123].txt` 匹配 `file1.txt`、`file2.txt`、`file3.txt` |
| `[!abc]` | 匹配括号内任意一个**除外**的字符 | `ls file[!0].txt` 匹配除 `file0.txt` 外的 `file?.txt` |
| `[[:class:]]` | 匹配指定字符类 | `[[:digit:]]` 匹配数字，`[[:alpha:]]` 匹配字母 |
| `{a,b,c}` | 匹配大括号内任意一个模式 | `ls {a,b,c}.txt` 匹配 `a.txt`、`b.txt`、`c.txt` |

### 实践建议

可以新建一个测试目录并创建一些文件，尝试上述各种 glob 模式来加深理解。

## Shell 引号详解

### 三种引号的区别

在 Bash 中，三种引号有不同的处理方式：

| 引号类型 | 说明 | 特殊字符处理 | 示例 |
|---------|------|-----------|------|
| **单引号** `'...'` | 强制字面量 | 所有字符都按字面含义处理 | `'$var'` 输出 `$var` 字符串 |
| **双引号** `"..."` | 允许展开 | 允许变量展开、命令替换、转义 | `"$var"` 会展开变量 |
| **$'...'** | ANSI-C 引号 | 支持转义序列（如 `\n`、`\t`） | `$'a\nb'` 输出换行 |

### 详细对比

**单引号的特点：**
- 禁止所有转义和展开
- 无法在单引号内包含单引号
- 示例：`echo '$HOME'` 输出 `$HOME`

**双引号的特点：**
- 允许变量展开（`$var`）
- 允许命令替换（`` `cmd` `` 或 `$(cmd)`）
- 允许某些转义序列（`\$`、`\"` 等）
- 示例：`echo "$HOME"` 会展开 `HOME` 变量

**ANSI-C 引号的特点：**
- 支持转义序列：`\n`（换行）、`\t`（制表符）、`\\`（反斜杠）等
- 支持八进制和十六进制：`\101` (A)、`\x41` (A)
- 示例：`echo $'line1\nline2'` 输出两行

### 输出包含 $、! 和换行符的字符串

使用 ANSI-C 引号可以优雅地解决这个问题：

```bash
echo $'Price: $100\nHistory: !bash\nNote: backslash \\'
```

或者使用单引号和双引号的组合：

```bash
echo '$ ! 
换行'
```

或使用 `printf`（更推荐）：

```bash
printf '%s\n' '$' '!' 'newline
after'
```

**最简洁的方案：**

```bash
echo $'$\n!\n'
```

这条命令输出：
- 字面量 `$`
- 换行符
- 字面量 `!`
- 换行符
